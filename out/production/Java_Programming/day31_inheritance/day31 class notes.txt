this anahtar sözcüğü: örnekleri ifade eder. geçerli sınıf

		this. : örnek değişkenleri ve örnek yöntemleri çağırır
		this() : kurucuyu çağırır


super anahtar sözcüğü: örneklere atıfta bulunur. ana sınıf

		super. : örnek değişkenleri ve örnek yöntemleri çağırır
		super() : kurucuyu çağırır

	5. Create a sub class of Vehicle named AirPlane

			Extra methods:

this keyword: reffers to the instances. current class

		this. : calls instance variables & instance methods
		this() : calls the constructor


super keyword: refers to the instances. parent class

		super. : calls instance variables & instance methods
		super() :  calls the constructor
